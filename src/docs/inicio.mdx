# Início

Blush é um Design System criado para facilitar, acelerar e padronizar o desenvolvimento de interfaces.
Forcene uma biblioteca de tokens de design (tipografias, cores, espaçamentos) e uma coleção de componentes reutilizáveis para criação de experiências consistentes e escaláveis.

Com o Blush, você pode:
- 💡 Simplificar decisões de design com tokens prontos.
- 🎨 Manter a identidade visual de forma consistente ao longo de projetos.
- ⚙️ Acelerar o desenvolvimento com componentes bem documentados e personalizáveis.

Código disponível no [Github](https://github.com/bruna-nunes/blush)
